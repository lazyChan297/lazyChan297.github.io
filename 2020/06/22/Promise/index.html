Promise

Promise是一个处理异步函数非常好的类,可以通过new Promise(handle)构造它的实例。
只有当你等待的函数有结果后，它还会进一步处理。
Promise一共有三种状态pending（等待中）,即没有回调结果时；resolve(成功)；reject(失败);
它的状态变化只能是 pending –> resolve 或者 pending –> reject;
一旦它的状态改变,就不可能回到原来的状态。顾名思义Promise

Promise的用法

接收一个handle函数作为入参数,handle里的代码会立即执行。通常是在handle中存放判断状态成功或失败的代码。
handle接收resolve和reject两个函数参数，分别处理成功与失败的回调
resolve和reject则是会等候一段时间后才会执行,例如说网络请求有返回后、倒计时结束后等等

Promise的链式调用

Promise支持实例链式调用.then方法。.then(res)的参数值取自上一个回调用法的返回值

Promise的返回值

Promise返回的是仍然是一个Promise实例，如果自定义返回了别的类型，也会将它封装成一个Promise实例
状态没有改变前的promise { PromiseStatus: pending, PromiseValue: undefined }
状态改变后的promise { PromiseStatus: rejected, PromiseValue: 1 }
