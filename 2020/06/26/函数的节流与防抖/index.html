函数的节流与防抖

发表于 2020-06-26 | 分类于 JavaScript

节流

节流也可以理解为减少开销，每隔一段时间才会让它执行。
就好比搜索输入框，为了减少向服务器发送多余的请求，就可以使用函数节流，让请求在一定时间段内只会发送一次
不管触发多少次，只会在一定时间间隔内执行一次

防抖

防抖常用于优化点击事件，防止频繁点击按钮发生过多请求
不管触发多少次，都会延迟一定时间执行一次
